# charemma-web

Static portfolio website for [charemma.de](https://charemma.de), served by Caddy with automatic HTTPS.

## Deployment

```
just deploy    # sync files + restart container
just sync      # sync files only
just status    # container status
just logs      # container logs
just restart   # restart container
```

## Local dev

```
just dev       # local server on http://localhost:8080
```
